# slopfiles

questionably useful claude code plugins

## Installation

Add the marketplace:

```bash
claude plugin marketplace add dexhorthy/slopfiles
```

Install the plugin:

```bash
claude plugin install clean-coder
```

## Plugins

### clean-coder
