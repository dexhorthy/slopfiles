# slopfiles

Claude Code plugins.

## Installation

Add the marketplace:

```bash
claude plugin marketplace add dexhorthy/slopfiles
```

Install the plugin:

```bash
claude plugin install clean-coder
```

## Plugins

### clean-coder

Professional coding assistant principles inspired by Uncle Bob.

**Skills:**

- `/say-hello` - Say hello world
- `/clean-coder` - Professional coding principles (invoke when frustrated)
